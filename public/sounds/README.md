# Custom Sounds

Drop replacement audio files here. Supported formats: MP3, OGG, WAV.

If any file is missing or fails to load the app falls back to a synthesized sound automatically.

## Objective neutralization

One file is picked at random each time an objective is neutralized.

| File | Plays when |
|------|-----------|
| `objective-complete-1.mp3` | Objective neutralized (random pick 1 of 6) |
| `objective-complete-2.mp3` | Objective neutralized (random pick 2 of 6) |
| `objective-complete-3.mp3` | Objective neutralized (random pick 3 of 6) |
| `objective-complete-4.mp3` | Objective neutralized (random pick 4 of 6) |
| `objective-complete-5.mp3` | Objective neutralized (random pick 5 of 6) |
| `objective-complete-6.mp3` | Objective neutralized (random pick 6 of 6) |

## Neutralize charge clicks

Played in sequence as you click to charge the neutralize action (pitch rises with each step).

| File | Plays when |
|------|-----------|
| `charge-1.mp3` | 1st click (lowest pitch) |
| `charge-2.mp3` | 2nd click |
| `charge-3.mp3` | 3rd click |
| `charge-4.mp3` | 4th click |
| `charge-5.mp3` | 5th click — triggers neutralization (highest pitch) |
